# WordMaster - 智能单词学习系统

> 一个基于Web的智能化英语单词学习应用，采用Vibe Coding开发模式构建

## 项目简介

WordMaster 是一款专为英语学习者设计的智能单词记忆应用。系统基于艾宾浩斯遗忘曲线原理，提供科学的单词学习、测试和复习功能，帮助用户高效掌握英语词汇。

## 核心功能

### 1. 单词学习
- 支持四级、六级、雅思、托福等多个词库
- 智能学习进度跟踪
- 单词发音播放（Web Speech API）
- 例句展示与理解

### 2. 单词测试
- 中译英、英译中、混合三种测试模式
- 智能选项生成
- 实时正确率统计
- 测试结果分析

### 3. 复习巩固
- 基于艾宾浩斯遗忘曲线的智能复习提醒
- 个性化复习计划
- 学习状态追踪（新词/学习中/已掌握/困难词）

### 4. 学习进度
- 学习数据统计可视化
- 近7天学习趋势图表
- 单词掌握情况分析
- 学习时长记录

### 5. 数据管理
- 本地数据持久化存储
- 学习数据导入/导出
- 设置自定义（每日目标、自动发音等）

## 技术栈

- **前端**: HTML5, CSS3, JavaScript (ES6+)
- **样式**: CSS Variables, Flexbox, Grid, Animation
- **存储**: LocalStorage API
- **语音**: Web Speech API
- **版本控制**: Git

## 使用说明

1. 直接在浏览器中打开 `index.html` 文件
2. 无需安装任何依赖，纯前端应用
3. 学习数据自动保存在浏览器本地存储中
